# Welcome to Concorde Elements Storybook 

The UI Elements library [(Gitlab)](https://gitlab.com/e-pilot/product/journey/journey-monorepo/-/tree/main/packages/concorde-elements) is part of journeys UI. 
It uses CSS Modules as a main CSS tool and [Material Symbols](https://developers.google.com/fonts/docs/material_symbols) for icons. 

This library is work in progress. Should you have any comments or notes, don't hesitate to contact team erubik.

The connections between journey blocks and UI components can be seen [here](https://airtable.com/appfZxYgwkev2HNEb/tblADKQvYqTMLvadM/viw70XnfDmWyKaLvo?blocks=hide).

The progress can be seen [here](https://e-pilot.atlassian.net/wiki/spaces/EO/pages/6755680343/Concorde+UI+components+progress+and+effort).
